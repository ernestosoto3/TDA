# Visual Direction

## Overview

TDA should use a modern, sharp, and sports-focused visual style. The interface should feel clean, organized, and easy to scan.

## Color Palette

A possible palette could include:

- White: `#FFFFFF`
- Light gray: `#F5F5F5`
- Near black: `#111111`
- Secondary gray: `#737373`
- Borders: `#E5E5E5`
- Red accent: `#F21D3F`
- Active status green: `#16B957`

Red should be used for active navigation icons, notifications, links, and important actions.

## Background Style

The main background should be white, with light gray used to separate sections.

Gradients, decorative textures, glass effects, and heavy shadows should be avoided.

## Typography

TDA should use **Inter** as its main font.

Bold weights should be used for titles, headlines, scores, and community names. Regular or medium weights should be used for descriptions and secondary information.

## Button Style

Buttons should use sharp rectangular shapes with no rounded corners.

Primary buttons could use a red background with white text. Secondary buttons could use a white or light-gray background with black text and a thin border.

## Card Style

Cards should have sharp corners, thin borders, clear spacing, and little or no shadow.

Feed images should use a consistent `4:5` aspect ratio.

## Icons

No emojis should be used.

Navigation, sports, settings, notifications, and general actions should use icons from one consistent icon library.

## Theme Direction

The first version should focus only on light mode.

Dark mode could be considered during future development.

## Visual References

Possible references include:

- **Instagram:** Feed structure and the “Para ti” and “Siguiendo” selector
- **Apple Music:** Search categories and image-based tiles
- **ESPN or FotMob:** Scores, game previews, and box scores
- **Reddit or Discord:** Public communities and reactions
- **Sports publications:** Headlines, photography, and editorial layouts

## Summary

The suggested direction is a light-mode interface using Inter, neutral backgrounds, red accents, sharp components, consistent icons, minimal shadows, and strong sports imagery.
